# har2img
Converts .h array images/image arrays to an image

There are 2 scripts/files which i will explain what it does

harray.py - Converts images to .h arrays / image arrays
har2img.py - Converts .h arrays or image arrays to an image 

```bash
git clone https://github.com/alvzrei8-bit/har2img

cd har2img

python3 file.py (will show usage automatically)
```
